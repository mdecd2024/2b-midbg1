# W5課程網誌心得

## 課程主旨

協同產品設計實習的第五週, 教學內容為先前影片整理、分組網誌的統整與 NX 零組件繪圖.

## 學習成果

分組任務

把教學影片分組整理和自評:  1.在教學影片上填上字幕, 另行傳到可以嵌入到各分組的網站上 2.從影片字幕中整理出逐字稿, 放在影片下方, 以 .txt 連結安排。
